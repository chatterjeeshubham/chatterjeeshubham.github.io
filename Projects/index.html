2023

  1. Estimation of Cognitive Load using Saccadic Features | M.Tech Thesis (Ongoing) Supervisor: Prof. Aurobinda Routray
    • Detected face using a CNN model and performed Pupil localization.
    • Estimated Saccades and Fixations using velocity-dispersion based threshold. Then Microsaccades estimated using a MAD-based Velocity Threshold in fixations. Binocular microsaccades selected based on temporal overlap. Outliers filtered out using RANSAC.
    • Cognitive load estimated using the saccadic features.

  2. Image and Text Compression using Huffman Encoding from Scratch March 2023
    • Built a Huffman encoding algorithm from scratch to compress text and image data. It assigns variable-length codes to symbols based on their frequencies, achieving efficient compression by using shorter codes formore frequent symbols.

  3. Audio classification with DCT as feature vector using Baysian classifier February 2023
    • Used Discrete Cosine Transform of 882-points window to extract features and then PCA to compress the features.
    • Implemented Baysian classifier to classify an audio clip with 87.8% accuracy.

  4. Feature Compression using PCA and SVD in Images from Scratch January 2023

2022

  1. Background Subtraction from Video using Extended Scale Invariant Local Binary Patterns October 2022
    • Extracted Colour Features and texture features from frames using Local Binary Patterns.
    • Using these features, each pixel is classified into background and foreground pixels.

  2. Detection of Premature Ventricular Contraction in ECG and S1 & S2 Valve Sound in PCG using ECG & PPG December 2022
    • For PVC detection in ECG, first location of QRS complex and then by using Form Factor we detected the PVC beats.
    • For PCG's S1 and S2 detection, we link QRS complex to S1, and then match Dicrotic notch in PPG for S2 in PCG.

  3. Summit Navigator: Image Histogram Local Maxima Extraction October 2022
    • The purpose of this project is to effectively extract local maxima of an image histogram. First image preprocessing and then initial peak detection is done. Introduced concept of Observability Index to identify dominant peaks, which are Local Maxima locations.
